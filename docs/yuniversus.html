Yuniversus з®Җд»Ӣ вҖӢ

дёәдәҶжҳҫзӨәе®Үжө©зі»еҲ—иҫ“е…Ҙжі•зҡ„зү№жңүеӯ—ж №пјҢжҲ‘дҪҝз”ЁеӨ©зҸ©еӯ—еә“е’ҢжҖқжәҗе®ӢдҪ“з”ҹжҲҗдәҶдё“й—Ёзҡ„еӯ—дҪ“пјҢе®ҡеҗҚдёә Yuniversus. еҚі гҖҢе®ҮгҖҚзҡ„жӢјйҹі Yu е’ҢгҖҢе®Үе®ҷ/еҗҲдёҖгҖҚзҡ„жӢүдёҒж–Ү Universus зҡ„еҗҲеҶҷгҖӮ

дёәдәҶж–№дҫҝжЈҖзҙўе’ҢжӢ“еұ•пјҢеӯ—ж №зҡ„жҺ’еёғдёҘж јжҢүз…§еӯ—ж №зҡ„еүҚдёүдёӘз¬”з”»гҖӮ

Yuniversus дёҚд»…еҢ…жӢ¬е®Үз Ғзҡ„зү№жңүеӯ—ж №пјҢд№ҹеҢ…жӢ¬дёҖдәӣеёёи§Ғзҡ„жұүеӯ—жһ„д»¶пјҢеҸҜд»Ҙз”ЁдәҺиЎЁзӨәе…¶д»–еҪўз Ғзұ»иҫ“е…Ҙжі•зҡ„жӢҶеҲҶгҖӮе®ғе’Ңжұүеӯ—иҮӘеҠЁжӢҶеҲҶзі»з»ҹжүҖе®ҡд№үз§Ғз”Ёеӯ—зҡ„жҳ е°„е…ізі»еңЁи§ҒжӯӨгҖӮ

Unicode еҢәеқ— вҖӢ

Yuniversus еҚ з”Ё Unicode з¬¬йӣ¶дҪҚйқўзҡ„з§Ғз”ЁеҢәпјҲPrivate Use Area / PUAпјүгҖӮе…·дҪ“ең°пјҢеҚ з”ЁдәҶ F400-F8FF зҡ„з ҒдҪҚгҖӮ

иҝҷдёӘдҪҚзҪ®зҡ„йҖүеҸ–пјҢеҹәдәҺжңҖе°ҸеҢ–еҗҢеӨ©зҸ©еӯ—еә“з¬¬йӣ¶дҪҚйқўпјҲP0пјүзҡ„еҶІзӘҒгҖӮиҝҷдёҖйғЁеҲҶпјҢеңЁеӨ©зҸ©еӯ—еә“дёӯеҹәжң¬дёәз©әгҖӮ

еӯ—ж №еҲҶеҢә вҖӢ

еӯ—ж №жҢүеүҚдёүдёӘз¬”з”»дёҘж јеҲҶеҢәгҖӮ

  1. жЁӘз«–ж’ҮзӮ№жҠҳпјҢжҢүйҰ–з¬”дёҖе…ұеҲҶдёә5дёӘеӨ§еҢәгҖӮ
  2. жҜҸеӨ§еҢәеҶҚж №жҚ®ж¬Ўз¬”еҲҶжҲҗ5е°ҸеҢәпјҢжҜҸдёӘе°ҸеҢәйў„з•ҷ32дёӘеӯ—ж №дҪҚзҪ®гҖӮ
  3. жҜҸе°ҸеҢәдёӯпјҢеүҚ16дёӘз¬¬дёүз¬”дёәз©әжЁӘз«–пјҢеҗҺ16дёӘз¬¬дёүз¬”дёәж’ҮзӮ№жҠҳгҖӮе…¶дёӯпјҢз«–гҖҒжҠҳд»ҺеҗҺеҗ‘еүҚжҺ’еҲ—гҖӮ
  4. жҜҸеӨ§еҢәйў„з•ҷдёүе°ҸеҢәеӨҮз”ЁгҖӮз¬¬дёҖе°ҸеҢәжҳҜеҚ•з¬”еҸҳеҪўгҖӮз¬¬дәҢдёүе°ҸеҢәжҳҜеүҚйқўж”ҫдёҚдёӢзҡ„еӯ—ж №гҖӮ
  5. жҜҸеӨ§еҢәдёҖе…ұеҸҜж”ҫзҪ®256дёӘеӯ—ж №гҖӮдёҖе…ұеҸҜж”ҫзҪ®1280дёӘеӯ—ж №гҖӮ
