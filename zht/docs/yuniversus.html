Yuniversus з°Ўд»Ӣ вҖӢ

зҲІдәҶйЎҜзӨәе®Үжө©зі»еҲ—ијёе…Ҙжі•зҡ„зү№жңүеӯ—ж №пјҢжҲ‘дҪҝз”ЁеӨ©зҸ©еӯ—еә«е’ҢжҖқжәҗе®Ӣй«”з”ҹжҲҗдәҶе°Ҳй–Җзҡ„еӯ—й«”пјҢе®ҡеҗҚзҲІ Yuniversus. еҚі гҖҢе®ҮгҖҚзҡ„жӢјйҹі Yu е’ҢгҖҢе®Үе®ҷ/еҗҲдёҖгҖҚзҡ„жӢүдёҒж–Ү Universus зҡ„еҗҲеҜ«гҖӮ

зҲІдәҶж–№дҫҝжӘўзҙўе’ҢжӢ“еұ•пјҢеӯ—ж №зҡ„жҺ’дҪҲеҡҙж јжҢүз…§еӯ—ж №зҡ„еүҚдёүеҖӢзӯҶз•«гҖӮ

Yuniversus дёҚеғ…еҢ…жӢ¬е®Үзўјзҡ„зү№жңүеӯ—ж №пјҢд№ҹеҢ…жӢ¬дёҖдәӣеёёиҰӢзҡ„жјўеӯ—ж§Ӣд»¶пјҢеҸҜд»Ҙз”Ёж–јиЎЁзӨәе…¶д»–еҪўзўјйЎһијёе…Ҙжі•зҡ„жӢҶеҲҶгҖӮе®ғе’Ңжұүеӯ—иҮӘеҠЁжӢҶеҲҶзі»з»ҹжүҖе®ҡзҫ©з§Ғз”Ёеӯ—зҡ„жҳ е°„й—ңдҝӮеңЁиҰӢжӯӨгҖӮ

Unicode еҚҖеЎҠ вҖӢ

Yuniversus дҪ”з”Ё Unicode з¬¬йӣ¶дҪҚйқўзҡ„з§Ғз”ЁеҚҖпјҲPrivate Use Area / PUAпјүгҖӮе…·й«”ең°пјҢдҪ”з”ЁдәҶ F400-F8FF зҡ„зўјдҪҚгҖӮ

йҖҷеҖӢдҪҚзҪ®зҡ„йҒёеҸ–пјҢеҹәж–јжңҖе°ҸеҢ–еҗҢеӨ©зҸ©еӯ—еә«з¬¬йӣ¶дҪҚйқўпјҲP0пјүзҡ„иЎқзӘҒгҖӮйҖҷдёҖйғЁеҲҶпјҢеңЁеӨ©зҸ©еӯ—еә«дёӯеҹәжң¬зҲІз©әгҖӮ

еӯ—ж №еҲҶеҚҖ вҖӢ

еӯ—ж №жҢүеүҚдёүеҖӢзӯҶз•«еҡҙж јеҲҶеҚҖгҖӮ

  1. жЁӘиұҺж’Үй»һжҠҳпјҢжҢүйҰ–зӯҶдёҖе…ұеҲҶзҲІ5еҖӢеӨ§еҚҖгҖӮ
  2. жҜҸеӨ§еҚҖеҶҚж №ж“ҡж¬Ўз¬”еҲҶжҲҗ5е°ҸеҚҖпјҢжҜҸеҖӢе°ҸеҚҖй җз•ҷ32еҖӢеӯ—ж №дҪҚзҪ®гҖӮ
  3. жҜҸе°ҸеҚҖдёӯпјҢеүҚ16еҖӢз¬¬дёүз¬”зҲІз©әжЁӘиұҺпјҢеҫҢ16еҖӢз¬¬дёүзӯҶзҲІж’Үй»һжҠҳгҖӮе…¶дёӯпјҢиұҺгҖҒжҠҳеҫһеҫҢеҗ‘еүҚжҺ’еҲ—гҖӮ
  4. жҜҸеӨ§еҚҖй җз•ҷдёүе°ҸеҚҖеӮҷз”ЁгҖӮз¬¬дёҖе°ҸеҚҖжҳҜе–®зӯҶи®ҠеҪўгҖӮз¬¬дәҢдёүе°ҸеҚҖжҳҜеүҚйқўж”ҫдёҚдёӢзҡ„еӯ—ж №гҖӮ
  5. жҜҸеӨ§еҚҖдёҖе…ұеҸҜж”ҫзҪ®256еҖӢеӯ—ж №гҖӮдёҖе…ұеҸҜж”ҫзҪ®1280еҖӢеӯ—ж №гҖӮ
